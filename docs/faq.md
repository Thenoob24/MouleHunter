# Questions Fréquentes

**Le bot peut perdre ?**  
Oui, si un autre joueur (ou bot) est plus rapide. Mais il est généralement dans le top 3 !

**Peut-on modifier la stratégie ?**  
Absolument ! Tout est dans `main.py`. Tu peux changer les poids des moules, valorisation des bonus, etc.

**Le bot apprend-il entre les parties ?**  
Pas encore (stratégie fixe), mais chaque partie est indépendante.

**Il ne se connecte pas, que faire ?**  
Vérifie :
- que le `.env` est bien à la racine
- l’IP et le port du serveur
- que le serveur de jeu est lancé
