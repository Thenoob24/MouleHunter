# Comment démarrer

### Étape 1 — Créer le fichier `.env`
À la racine du projet, crée un fichier nommé `.env` :

```env
SERVEUR_IP=127.0.0.1
SERVEUR_PORT=1337
NOM_EQUIPE=LesMoulesFrites
```

### Étape 2 : Démarrer le serveur java
Pour ce faire rendez vous sur le github: https://github.com/synave/peche_aux_moules 
pour plus d'information

### Étape 3 : installer les dépendances
Ouvrez un terminal et taper: 
````powershell
pip install -r requirements.txt
````

### Étape 4 : Lancer le client
Ouvrez un terminal et taper: 
````powershell
python main.py
````

et voilà amusez vous bien !
